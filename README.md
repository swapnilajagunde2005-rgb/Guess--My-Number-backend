# 🎯 Guess My Number — Full-Stack Edition

An interactive numbers-guessing game built to master frontend DOM manipulation and bridge it seamlessly with a backend relational database.

## ✨ Features

- **Dynamic UI Updates:** Real-time feedback using vanilla JavaScript DOM methods.
- **State Management:** Tracks current scores, dynamically calculates high scores, and manages active win/loss states.
- **Asynchronous Data Bridge:** Utilizes the JavaScript `Fetch API` to send winner data as JSON payloads without reloading the page.
- **Secure Backend Architecture:** Built with a PHP processing engine that uses secure **PDO Prepared Statements** to safely execute queries and mitigate SQL Injection risks.
- **Persistent Leaderboard:** Saves records into a structured MySQL database.

## 🛠️ Tech Stack & Key Concepts Used

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend & Database:** PHP (Stream IO), MySQL (Data Constraints)
